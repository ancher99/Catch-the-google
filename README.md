# Игра Cath the google (pet-проект)

## Описание проекта 
 Веб-приложение, игра «Cath the google». Основная цель создания игры состоит в применение патерна **Observer** для отрисовки отдельных узлов в DOM дереве где произошло изменения для оптимизации приложения на чистом **JavaScript**.
 В проекте используеться модульный тип проектирования.
