# Игра Cath the google (pet-проект)

## Описание проекта 
Разработано веб-приложение — игра "Catch the Google". Ключевой технической задачей проекта стала оптимизация производительности приложения за счет реализации паттерна **"Наблюдатель" (Observer)**. Этот подход позволил минимизировать манипуляции с **DOM-деревом**, обеспечивая перерисовку только тех узлов интерфейса, в которых произошли изменения. Проект реализован на чистом **JavaScript** с применением принципов модульного проектирования.
